# Exploratory-Data-Analysis (EDA) on Titanic dataset 
Exploratory Data Analysis (EDA) on the Titanic dataset using Python, Pandas, Matplotlib, and Seaborn.

## Objective
The objective of this task is to perform Exploratory Data Analysis (EDA) on the Titanic dataset to identify patterns, trends, relationships, and insights using statistical summaries and data visualizations.

## Tools & Libraries Used

- Python
- Jupyter Notebook
- Pandas
- Matplotlib
- Seaborn

## Dataset

- **Dataset:** Titanic Dataset (CSV)

## Steps Performed

1. Imported the required libraries.
2. Loaded the Titanic dataset.
3. Explored the dataset using:
   - 'df.head()'
   - 'df.info()'
   - 'df.describe()'
   - 'df.value_counts()'
4. Checked and handled missing values.
5. Checked for duplicate records.
6. Performed Exploratory Data Analysis using:
   - Histograms
   - Boxplots
   - Scatter Plot
   - Correlation Heatmap
   - Pairplot
   - Countplots
7. Added observations after each visualization.
8. Summarized the key findings.

## Key Insights

- Most passengers were between 20 and 40 years old.
- The Fare feature is positively skewed and contains outliers.
- Female passengers had a higher survival rate than male passengers.
- First-class passengers had a better survival rate than lower-class passengers.
- Passenger Class showed a negative correlation with survival, while Fare had a weak positive correlation with survival.
